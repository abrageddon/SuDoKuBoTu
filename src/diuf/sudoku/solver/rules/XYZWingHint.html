XYZ-Wing

The three cells {0}, {1} and {2} form an XYZ-Wing pattern on the value {3}. Regardless of the correct value of the cell {0} ({3}, {4} or {5}), either the cell {0} contains the value {3}, or one of the two cells {1} or {2} will have the value {3} as its only remaining possible value. It follows that one of the three cells {0}, {1} or {2} must contain the value {3}.

Other occurances of the value {3} can therefore be removed from any cells sharing a row, column or block with all the three cells.
